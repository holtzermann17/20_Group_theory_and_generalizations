length function


Let G be a group. A length function on G is a function L:G→ℝ+ satisfying:

L⁢(e) = 0,
L⁢(g) = L⁢(g-1),∀g∈G,
L⁢(g1⁢g2) ≤ L⁢(g1)+L⁢(g2),∀g1,g2∈G.
